Dynamic Yoga Site

The Wellness Site was created as part of my 2nd year university project.

The site allows users to sign up for their own accounts, create custom routines, add custom poses, leave reviews for poses, favourite poses and explore poses from different categories of yoga.

The site utilises HTML, CSS, JavaScript, PHP, Smarty, MySQL and phpMyAdmin.

Login & Registration System

Users can register for an account to access all site features including adding poses, adding routines and gaining points in community leaderboards.

Passwords stored in the database are hashed and salted for a higher level of security.

Community Leaderboards

The leaderboards feature adds an engaging community element to the website. When uploading a pose, leaving a review, adding a pose to their favourites or adding a routine, the user is given a 'yoga point'.

These points build up over time and the top contributors to the site are recognised on the leaderboards page.
